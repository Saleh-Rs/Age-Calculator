# Age Calculator

A simple age calculator built with Python(Tkinter) that calculates the exact age based on birth date.


# Features

- simple and user-friendly GUI
- Calculates age in years, month, and days
- Handles invalid dates and input errors
- Lightweight and fast


# Built With 

- Python 3
- Tkinter (GUI library)


# How to Run

1. Make sure Python is installed on your system
2. Download or clone the project folder
3. Open terminal in the project folder
4. Run:
```bash 
python main.py
```
